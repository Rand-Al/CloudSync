Professional SaaS landing page theme with modern design, contact forms, and comprehensive customization options.

== Description ==

CloudSync is a premium-quality WordPress theme specifically designed for SaaS companies, startups, and digital service providers. Built with conversion optimization in mind, it features a clean, modern design that helps turn visitors into customers.

**Key Features:**
* ✅ One-page landing design optimized for conversions
* ✅ Fully responsive and mobile-optimized
* ✅ Comprehensive WordPress Customizer integration
* ✅ Built-in contact form with AJAX submission
* ✅ Essential SEO optimization (compatible with SEO plugins)
* ✅ Fast loading and performance-optimized
* ✅ Font Awesome icons included
* ✅ Google Fonts integration
* ✅ Cross-browser compatible

**Sections Include:**
* Hero section with call-to-action
* Features showcase
* How it works process
* Pricing plans
* Contact form
* Footer with customizable content

== Installation ==

1. Upload the theme files to `/wp-content/themes/cloudsync/`
2. Activate the theme through the 'Appearance > Themes' screen in WordPress
3. Go to 'Appearance > Customize' to configure your landing page content
4. Customize sections, colors, and content through the Theme Customizer

== Customization ==

All theme content can be customized through **Appearance > Customize > CloudSync Theme Options**:

* **Hero Section**: Title, description, buttons, and floating cards
* **Features**: 6 customizable feature cards with icons and descriptions  
* **How It Works**: 3-step process section
* **Pricing**: 3 pricing plans with features and buttons
* **Contact**: Contact information and form labels
* **Footer**: Company info, social links, and legal pages

== Contact Form ==

The theme includes a built-in AJAX contact form that:
* ✅ Works without plugins (but compatible with Contact Form 7)
* ✅ Includes spam protection with WordPress nonces
* ✅ Sends emails to admin email address
* ✅ Provides user feedback without page reload
* ✅ Includes loading states and error handling
